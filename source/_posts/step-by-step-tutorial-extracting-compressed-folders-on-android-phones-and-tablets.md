### Key Takeaways

* To open a ZIP file, download a file manager app like "Files by Google" from the Google Play Store, locate the ZIP file, and tap "Extract" to unzip the files.
* To create a ZIP file on Android, install a third-party app like "RAR" from Google Play, select the files you want to zip, tap the zip icon, give the ZIP file a name, and choose a location to save it.

 It's pretty common to zip files and unzip folders on a computer, but you might also have to do it on your Android phone or tablet. This used to be an annoying task, but it's much easier nowadays. We'll show you how.

 It's not uncommon to run across ZIP files on Android---especially if you [connect a USB drive](https://extra-information.techidaily.com/optimize-iphone-cinematography-ultimate-capture-additions/). Every Android phone comes with a file manager app, but they're typically barebones and can't open ZIP files. Thankfully, there are a number of apps on the Google Play Store that can do it for free.

 One app that we like is "[Files by Google](https://www.anrdoezrs.net/links/3607085/type/dlg/sid/UUhtgUeUpU211532/https://play.google.com/store/apps/details?id=com.google.android.apps.nbu.files)." Not only is this a good app for uncompressing ZIPs, but it's also a good all-around file manager that can also be used to [free up space on your smartphone](https://facebook-videos.techidaily.com/updated-recorded-reality-facebook-live-2023/). The app may have come pre-installed on your Android device.

##  How to Create a ZIP File on Android

 Android doesn't have a built-in way to create ZIP files. You'll need to use a third-party app that can do this. There are many out there, but we recommend and will be using "[RAR](https://www.anrdoezrs.net/links/3607085/type/dlg/sid/UUhtgUeUpU211532/https://play.google.com/store/apps/details?id=com.rarlab.rar&hl=en&gl=US)" for this. You can install it from Google Play.

 Open RAR and locate the files you'd like to zip. Check their boxes to select them.

![Select the files.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2023/10/screenshot_20231005-164305.png) 

 After you've selected all the files, tap the zip icon---stack of books with a plus icon---in the top toolbar.

![Tap the zip icon.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2023/10/screenshot_20231005-164313.png) 

 Next, give the ZIP file a name and select "ZIP" as the extension.

![Give the zip file a name and select "ZIP" as the extension.](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2023/10/screenshot_20231005-164342.png)
